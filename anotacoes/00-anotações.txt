5 - Functions.php x Menus
para o wordpress reconhecer seu site possui os menus:
a). criar no aquivo functions, o seguintes trecho de códigos:
function nomeDaFuncao_setup(){
    add_theme_support( 'post-thumbnails' );
    add_theme_support( 'title-tag' );

    register_nav_menus(
        array(
            'primary' => 'Menu Principal'
        )
    );
 }
 add_action('after_setup_theme', 'nomeDaFuncao_setup');
